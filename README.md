
SpaceMedia test app
=======
Тестовое задание для Спейс Медиа.

0) Разработать на QT программу, которая будет делить экран на произвольные части
(настройки задаются в xml/json формате) и выводит в каждую из них произвольный
контент. Там же указывается та часть, из которой воспроизводить звук.

1) Синий - вывод 3х MP4 файлов по порядку (хранятся в папке с приложением)

2) Желтый - вывод данных о погоде - статичная картинка на фоне, вывод данных о погоде
из https://export.yandex.ru/weather-ng/forecasts/26898.xml

3) Зеленый - обновление и вывод контента из Instagram (аккаунт andreysm32 /
12345678zx) выводить для примера последний пост с тегом #wow

4) трансляция IPTV : к примеру http://109.111.184.197:4022/udp/239.211.211.20:1234


======

Настройки хранятся в data/config.txt

Можно создавать любое количество виджетов и задавать им произвольный размер и позицию на экране, как в пикселях, так и в процентах.

Сейчас в плейлисте только 2 mp4 файла, можно добавить сколько угодно в настройках.

Источник звука задается в массиве sound - можно задавать громкость каждому виджету по его id. По дефолту - звук отключен.
